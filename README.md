# Sociolla Interview - Frontend

## Features
- Show a popup screen (bottom sheeet) when a promo button is clicked
- Display a list of promos from the backend
- Copy promo code into clipboard

## Technology Stack
- React Native
- Fetch API for backend communication

## Setup & Run
1. Clone the repository:
```bash
git clone [frontend-repo-link]
cd [frontend-folder-name]
```
2. Install dependencies:
```bash
npm install
```
4. Run the app:
```bash
npx react-native run-android
# or
npx react-native run-ios
```
